System level. Avoid collisions.

General
    command space // spotlight
    option  space // iTerm (defined in iTerm Preferences)
    command space shift // siri

    command \ // alternate app switcher
    command shift \ // invert colors.

    fn fn // dictation

    command [6-9|0] // manage side menus in applications
    command shift / // show help menu

    shift esc // escape search and suggestion menus.

Text Editors and Terminals
    command []       // change indent
    command [] shift // change tabs

    Split Panes
        command d       // split pane right
        command d shift // split pane down

        command shift j|k|h|l   // navigate split panes

        command control <arrow key> // resize split pane.

    command option [ OR ]       // code fold single
    command option shift [ OR ] // code fold all

    iTerm: command enter // toggle non-native full-screen.

VS Code

    command p // open file
    command o  // find symbol

    command shift p // search for command
    command shift o // go to symbol definition.

    option o // toggle between header and source.

    command option control b CORD TO t // toggle bookmark
    command option control b CORD TO l // list bookmarks
    command y // show debug window
    command shift y // show debug console
    command shift r // start debugging
    Reminders

Magnet

BetterTouchTools
    fn f1  // Open iTerm
    fn f2  // launch Unity Hub
    fn f3  // Show menu bar in context menu. Shows menu bar like a right click. Great for large displays.
    fn f8  // sleep computer
    fn f12 // Center cursor on screen
    command \ // app switcher + invert colors.
    4 finger click // app switcher
    5 finger click // show menubar in context menu
    5 finger tap // f10 (bartender items)
    TipTap Left (2 Finger Fix) -> command w
    Mouse Fwd Button // show menubar in context menu
